## `starwars hello`

Say hello for someone, or for everyone!

```
USAGE
  $ starwars hello

OPTIONS
  -n, --name=name  Name to say hello

DESCRIPTION
  If you want to say hello for someone, try something like this two examples:

  $ starwars hello -n=Eduardo
  $ starwars hello --name=Eduardo
```
